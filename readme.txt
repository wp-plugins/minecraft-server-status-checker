== Description ==

This plugin will detect and show the Minecraft Server Status. Works with any kinds of server.

== Installation ==

Use WordPress' Add New Plugin feature, searching "Minecraft Server Status Checker (MSSC)", or download the archive and:

1. Upload `minecraftsv` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets under Appearance and start playing with our widget!

== Frequently Asked Questions ==

= The plugin doesn't work, why? =

Try de-activate and activate it again
